## Sobre o Projeto

O **Descubra Recife** é uma aplicação front-end desenvolvida em React com o objetivo de apresentar a cidade do Recife de forma interativa, moderna e acessível. A plataforma centraliza informações sobre pontos turísticos, locais culturais e dados urbanos, oferecendo uma experiência visual envolvente para exploração da cidade.

A proposta é transformar dados e informações em uma interface intuitiva, permitindo que usuários descubram novos lugares e compreendam melhor o ambiente urbano.

### Funcionalidades da versão atual

- Visualização de pontos de interesse da cidade  
- Interface moderna com foco em experiência do usuário  
- Navegação fluida entre seções  
- Layout responsivo para diferentes dispositivos  
- Componentização com React  
- Possível integração com dados externos/APIs
